# NavSkill Learning

NavSkill Learning is an educational website built using **HTML and CSS**.
The design was inspired by layouts found on **Pinterest**, and the website was created from scratch to practice frontend development and layout design.

## Features

* Responsive navigation bar
* Hero section with call-to-action buttons
* Company logo showcase section
* Clean and modern layout

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was created as a **practice project** to improve my skills in HTML, CSS, and webpage layout design.
